----- 01 TASK -----
-- Задание --
Нужно создать объект, описывающий автомобиль с любыми произвольными свойствами.
- Запросить у пользователя ввод названия этого свойства.
- Затем вывести значение нужного свойства в консоль.
- Если такое свойство отсутствует, то вывести сообщение "Not found"

-- Решение --
- Создаем объект с 3-мя свойствами
- Запросим у пользователя ввод имени свойства, и запишем это свойство в переменную input
Мы получим в отдельную переменную значение по этому имени свойства. То есть пользователь ввел
имя свойства в переменную input и мы его сохранили.
- Далее мы берем объект auto и проверяем поле с именем которое хранится в переменной input
!!! Обратите внимание на то, что мы сделали это через квадратные скобки потому, что через точку это было
сделать нельзя. Иначе бы компьютер искал название input в списке полей объекта!!!
- А теперь проверим, если такое поле есть, то мы выведем его значение в консоль, в противном случае
если такое поля нет, то значение будет не найдено и в переменную value запишется undefined

----- Резюме программы -----
В случае, если у нас значение, которое ввел пользователь совпадает с названием одного из полей,
то есть в переменной value хранится какое-то значимое значение не undefined , то переменная будет
преобразована в true , и мы выведем сообщение в консоль с соответствующим значением поля.
- В противном случае, если в переменной value будет значение undefined , то тогда условие не выполнится
и в консоли появится сообщение Not found

----- Запустим программу в веб-браузере -----
- Появляется поле для ввода данных.
(Итак мы знаем, что у нашего объекта есть 3 свойства):
 model,
 color,
 year

- В веб-браузере, в выпадающем окне введем model и получаем в консоль значение audi (правильное значение)
- Обновляем браузер и теперь вводим year, получаем в консоль 2012 (правильное значение)
- Теперь введем свойство, которого нет. Например, введем type, получаем в консоль Not found
